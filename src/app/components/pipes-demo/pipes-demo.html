Let's Learn Angular Pipes

uppercase , lowercase, titlecase pipe:
Name without any pipe: {{myName}}
Name with uppercase pipe: {{myName | uppercase}}
Name with lowercase pipe: {{myName | lowercase}}
Name with titlecase pipe: {{myName | titlecase}}
Currency pipe:
mySal without any pipe: {{mySal}}
mySal with currency pipe: {{mySal | currency}}
mySal with currency pipe: {{mySal | currency : 'USD' : 'symbol'}}
mySal with currency pipe: {{mySal | currency : 'USD' : 'code'}}
mySal with currency pipe: {{mySal | currency : 'INR'}}
mySal with currency pipe: {{mySal | currency : '€'}}
mySal with currency pipe: {{mySal | currency : '元'}}
Date pipe:
{{today}}
{{today | date}}
{{today | date : 'shortDate'}}
{{today | date : 'mediumDate'}}
{{today | date : 'longDate'}}
{{today | date : 'fullDate'}}
{{today | date : 'shortTime'}}
{{today | date : 'mediumTime'}}
{{today | date : 'longTime'}}
{{today | date : 'fullTime'}}
{{today | date : 'dd / MMM / yyyy'}}
{{today | date : 'dd / MMMM / yyyy' | lowercase}}
{{today | date : 'dd / MMMM / yyyy' | uppercase}}
{{today | date : 'dd / MMMM / yyyy' }}
Number/Decimal & percent pipe:
{{ 12.111222333 | number: "3.2-5" }}
{{ 2.5 | number: "3.2-5" }}
{{ 2.5 | percent }}
{{ 0.7 | percent }}
{{ 0.064 | percent }}
{{ 0.065 | percent }}
json pipe:
{{user | json }}
keyvalue pipe:
@for(entry of user | keyvalue;track $index){
{{entry.key}}---{{entry.value}}
}
slice pipe:
{{arr}}
{{arr | slice : 2}}
{{arr | slice : 2 : 5}}
async pipe:
Observable:: {{num}}
Subscribe the Observable:: {{num | async}}
Custom pipe (Remaining Char):
Remaining Char: {{msg | remaining}}
Remaining Char: {{msg2 | remaining : 200}}
